== Overview

=== 인증

모든 인증이 필요한 API는 요청 헤더에 Bearer 토큰을 포함해야 합니다.

[source,http]
----
Authorization: Bearer {accessToken}
----

=== 공통 응답

==== 성공 응답

성공 응답은 `data` 필드 안에 실제 데이터가 담겨 있습니다.

[source,json]
----
{
  "data": {
    "accessToken": "eyJhbGc..."
  }
}
----

==== 에러 응답

에러 응답은 RFC 7807 ProblemDetail 형식을 따릅니다.

[source,json]
----
{
  "type": "about:blank",
  "title": "Bad Request",
  "status": 400,
  "detail": "유효하지 않은 토큰입니다.",
  "instance": "/api/v1/auth/refresh",
  "properties": {
    "timestamp": "2026-04-28T20:00:00",
    "code": "INVALID_TOKEN"
  }
}
----
