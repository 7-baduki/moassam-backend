=== 좋아요 정보 조회

게시글 조회 시 현재 로그인 사용자의 좋아요 여부와
게시글의 총 좋아요 수를 함께 반환합니다.

[source,json]
----
{
  "likeCount": 10,
  "isLiked": true
}
----

|===
|필드명|설명

|likeCount
|게시글의 총 좋아요 수

|isLiked
|현재 로그인 사용자의 좋아요 여부

|===
