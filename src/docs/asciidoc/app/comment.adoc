== 댓글 API

=== 댓글 등록

게시글에 댓글을 등록합니다.

댓글 내용(content)은 필수입니다.

operation::comment/create-comment[snippets='http-request,http-response']


=== 댓글 조회

게시글에 등록된 댓글을 조회합니다.

댓글은 게시글 하위 리소스이므로 postId와 commentId가 모두 필요합니다.

operation::comment/get-comment[snippets='http-request,http-response,response-fields']


=== 댓글 수정

댓글 내용을 수정합니다.

본인이 작성한 댓글만 수정할 수 있습니다.

operation::comment/update-comment[snippets='http-request,http-response']

=== 댓글 삭제

댓글을 삭제합니다.

본인이 작성한 댓글만 삭제할 수 있습니다.

operation::comment/delete-comment[snippets='http-request,http-response']
