== 관찰일지 API

=== 관찰일지 생성

교사가 입력한 관찰 내용을 바탕으로 관찰일지를 생성합니다.

대상 연령(age), 관찰 영역(sectionTypes), 관찰 내용(situation)은 필수입니다.

operation::observation/create-observation[snippets='http-request,request-fields,http-response']


=== 관찰일지 재생성

기존 관찰일지의 관찰 내용과 관찰 영역을 기준으로 관찰일지를 다시 생성합니다.

기존 제목, 요약, 영역별 관찰 내용이 새로 생성된 내용으로 변경됩니다.

operation::observation/regenerate-observation[snippets='http-request,http-response,response-fields']


=== 관찰일지 단건 조회

관찰일지 ID로 관찰일지 상세 내용을 조회합니다.

제목, 요약, 영역별 관찰 내용을 함께 반환합니다.

operation::observation/get-observation[snippets='http-request,http-response,response-fields']


=== 관찰일지 목록 조회

사용자가 생성한 관찰일지 목록을 조회합니다.

cursor와 size를 사용하여 커서 기반 페이지네이션을 지원합니다.

operation::observation/getList-observation[snippets='http-request,http-response,response-fields']


=== 관찰일지 삭제

관찰일지를 삭제합니다.

관찰일지에 포함된 영역별 관찰 내용도 함께 삭제됩니다.

operation::observation/delete-observation[snippets='http-request,http-response']
