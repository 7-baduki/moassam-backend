== 게시글 API

=== *자유게시판* 게시글 등록

자유게시판 게시글을 등록합니다.

자유게시판은 말머리(headTag)가 필수이며, 연령(age)과 자료 유형(resourceType)은 사용하지 않습니다.

operation::post/create-free[snippets='request-parts,request-part-request-fields']

==== HTTP Request

[source,http]
----
POST /api/v1/posts HTTP/1.1
Content-Type: multipart/form-data
Host: localhost:8080

--boundary
Content-Disposition: form-data; name=request
Content-Type: application/json

{
  "category": "FREE",
  "age": null,
  "resourceType": null,
  "headTag": "WORRY",
  "title": "이직 준비중인데 고민이 있습니다.",
  "content": "게시글 내용"
}

--boundary
Content-Disposition: form-data; name=files; filename=test.pdf
Content-Type: application/pdf

(file binary)
----

operation::post/create-free[snippets='http-response,response-fields']

=== *자유게시판* 게시글 수정

자유게시판 게시글을 수정합니다.

기존 파일 삭제가 필요한 경우 deleteFileIds에 삭제할 파일 ID 목록을 전달합니다.

수정 요청은 최종 게시글 상태 전체를 전송합니다.

operation::post/update-free[snippets='request-parts,request-part-request-fields']

==== HTTP Request

[source,http]
----
PATCH /api/v1/posts/1 HTTP/1.1
Content-Type: multipart/form-data
Host: localhost:8080

--boundary
Content-Disposition: form-data; name=request
Content-Type: application/json

{
  "category": "FREE",
  "age": null,
  "resourceType": null,
  "headTag": QUESTION",
  "title": "수정된 제목",
  "content": "수정된 내용",
  "deleteFileIds": [10]
}

--boundary
Content-Disposition: form-data; name=files; filename=new-file.pdf
Content-Type: application/pdf

(file binary)
----

operation::post/update-free[snippets='http-response,response-fields']
